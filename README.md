# Kişisel Dijital Portfolyo Projesi

Bu proje, Kapadokya Üniversitesi Web Tasarımı dersi kapsamında, HTML5, CSS3 ve JavaScript kullanılarak geliştirilmiş responsive (mobil uyumlu) bir kişisel portfolyo sitesidir.

* **Bölüm:** Bilgi Güvenliği Teknolojisi

## 🚀 Proje Özellikleri
* **Semantik HTML5:** Modern etiket yapısı (header, section, article, footer).
* **CSS3 Flexbox:** Esnek ve hizalı sayfa düzeni.
* **Responsive Tasarım:** Mobil ve tablet cihazlarla tam uyumlu görünüm.
* **JavaScript İnteraktivitesi:** İletişim formu doğrulama ve kullanıcı geri bildirimi.
* **Paralaks Efektleri:** Görsel derinlik katan arka plan geçişleri.
